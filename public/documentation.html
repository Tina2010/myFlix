myFlix - Api Documentation

The following page provides all necessary context to understand how to API works and how it is possible to run the API.

Description:

The myFlix API offers a way to provide users with access to information about different movies, directors, and genres. You can take a first look at this APIs work in my application: myFlix.

POST

    }],

    Director: [{

        type: mongoose.Schema.Types.ObjectId, ref: 'Director'

    }],

    Actors: [String],

    ImagePath: String,

    Featured: Boolean
